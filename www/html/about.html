About Webserv

Project Overview

Webserv is a custom HTTP/1.1 web server implementation built from scratch in C++98. It demonstrates deep understanding of network programming, HTTP protocol, and non-blocking I/O operations.

Technical Stack

  • Language: C++98
  • Protocol: HTTP/1.1 (RFC 2616)
  • I/O Model: Non-blocking with poll()
  • CGI: RFC 3875 compliant

Architecture

The server follows a modular architecture with clear separation of concerns:

  • Configuration Module: Parse NGINX-style config files
  • Server Core: Socket management and event loop
  • Protocol Module: HTTP parsing and response formatting
  • Application Module: Request routing and handlers
  • Utility Module: String/file operations, logging
