Login Test (Session & Cookies)

Test session management with cookies.

How it works:

  • Submit a username to create a session
  • Server generates a unique session ID
  • Session ID stored in cookie (SESSID)
  • You'll be redirected to dashboard
  • Check cookies in browser DevTools (F12 â†’ Application/Storage)

Test with cURL:

# Login and save cookies
curl -X POST http://localhost:8062/do_login \
  -d "username=testuser" \
  -c cookies.txt -v

# Access dashboard with saved session
curl http://localhost:8062/dashboard \
  -b cookies.txt

# Access profile
curl http://localhost:8062/profile \
  -b cookies.txt

# Logout
curl http://localhost:8062/logout \
  -b cookies.txt -v
